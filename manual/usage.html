How to Run the Code

Command-Line Options

DeCE runs in three different modes depending on how command-line options are provided.

The first one is like below: an ENDF-6 data file is only provided without any command-line options.

 % dece  ENDF_in.DAT

DeCE will be in an interactive mode, and print out a section only when user indicates so. DeCE continues running until you type quit, end or exit.

The second one is to use an option of -o File_Name.

 % dece  -o ENDF_out.DAT ENDF_in.DAT

Again DeCE will be in the interactive mode. Once DeCE processed all the instructions provided, and read quit command, it writes the whole ENDF-6 formatted data on the ENDF_out.DAT file.

DeCE can be used for renumbering only, without any numerical manipulation. In this case, typing the quit command at each time is a bit anoying. There is a command line option -q, which tells DeCE to exit right away and reformat the file.

 % dece  -q -o ENDF_out.DAT ENDF_in.DAT

The last mode is to tabulate a specific section in the ENDF file.

 % dece -f  MF  -t  MT   ENDF_in.DAT

These options -f and -t tells DeCE to extract that MF/MT section, and print the data in a human readable format. Note that this function has not been completed yet. Some Mf/MT sections cannot be processed.

More Options

There are some more command-line options, DeCE recognizes:

-r
Print thermal cross sections calculated from resonance parameters.
-e En
Print a cross section at a given energy En.
-x
Scan ENDF-6 file, and make a summary in XML.
-v
Verbose mode when DeCE scans an input file.
-h
Print help.
